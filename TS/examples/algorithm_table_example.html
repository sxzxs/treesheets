初始化 open_heap = [(h(S), 0, S)] g[S]=0; came_from={}; closed={} open_heap 按 f = g + h 排序
主循环 while open_heap 非空 pop 最小 f；判断 goal；扩展邻居

while open_heap 非空 current = pop_min_f(open_heap) 总是扩展当前估价最小的格子
if current == goal return reconstruct_path(came_from, current) 找到最短路径，提前结束
if current in closed continue 跳过堆里的旧副本
assign current not closed closed.add(current) 该格子以后不再扩展
for 每个可达邻居 relax(current, neighbor) 展开看松弛逻辑
块 条件 语句 效果
for neighbor in four_neighbors(current) 依次检查右、左、下、上 每个邻居独立判断
if neighbor 越界 or grid[neighbor] == '#' continue 墙和地图外不入堆
assign tentative_g = g[current] + 1 步长固定为 1 候选路径长度
if tentative_g < g.get(neighbor, INF) came_from[neighbor] = current 发现更短路径
then 更新 g/f 并 push open_heap f = tentative_g + Manhattan(neighbor, goal) 后续优先弹出低 f 节点
else 已有路径更短或相等 ignore neighbor 不修改父指针
invariant closed 中节点的 g 已经最优 open_heap 保存候选前沿 came_from 可回溯路径
after while open_heap 空 return NO_PATH 目标不可达
