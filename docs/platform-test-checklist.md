# 平台与界面验证清单

这些项目依赖真实窗口系统、输入法或平台文件监控，不适合在当前无窗口单元测试里强行模拟。CI 已覆盖 Linux、Windows、macOS 的构建和无窗口测试；下面项目作为发布前手工验证清单保留。

- macOS 输入法组合输入：在中文或日文输入法下编辑单元格，确认候选、提交、撤销不会破坏光标位置。
- Windows 文件监控：外部替换式保存和原地保存都只触发一次重载提示。
- macOS 文件监控：快速连续保存不会漏提示，不会重复提示。
- Linux 文件监控：替换式保存、原地保存、删除文件都能进入预期提示路径。
- 高分屏缩放：字体、网格线、图片在 100%、150%、200% 缩放下无明显错位。
- 多显示器：窗口位置保存和恢复不会落到不可见区域。
- 暗色模式：默认文本、网格线、选区和标签颜色保持可读。
- 菜单、工具栏、快捷键：常用编辑、保存、导入导出命令可触发并且状态栏提示合理。
- 打印预览和分页：基础文档能打开预览，分页缩放不会崩溃。
